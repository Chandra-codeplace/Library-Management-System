# 📚 Library Management System

## 🔹 Project Overview
The Library Management System is a web-based application developed using ASP.NET Web Forms. It helps manage books, students, and borrowing activities efficiently.

---

## 🚀 Features

- 📖 View Available Books
- 📥 Issue Books to Students
- 📤 Return Books
- 👨‍🎓 Student Login & Dashboard
- 👨‍💼 Admin Login & Management
- 🔍 Search Books
- 📊 Track Borrowed Books
- Automatic Email alert system

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, Bootstrap5
- Backend: C# (ASP.NET Web Forms)
- Database: SQL Server
- IDE: Visual Studio 2022

---

## 🗂️ Database Structure

Table includes:
- Book ID
- Title
- Author
- Publisher
- ISBN
- Publication Year
- Availability Status
